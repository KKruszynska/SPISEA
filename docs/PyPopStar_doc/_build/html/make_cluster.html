ResolvedCluster¶

class Isochrone(params)

Simulate a star cluster.

Parameters:
  • iso – The theoretical cluster isochrone (Isochrone Object)
  • imf – The initial mass function (IMF Object)
  • cluster_mass – The cluster mass, in M_sun
  • ifmr – The initial-final mass relation (IFMR Object)
  • set_random_seed – If true, then set random seed of IMF sampling
  • verbose – If true, print out additional info
